# FitCart AI Web

React frontend placeholder.

## Responsibility

- Catalog browsing
- Search and filters UI
- Product comparison UI
- Recommendation and document upload flows
- Displaying AI-generated summaries and explanations

## Suggested next implementation steps

1. Choose build tool: Vite is the simplest default.
2. Create feature-based folders under `src/`.
3. Keep API clients thin and typed.
4. Treat the frontend as presentation plus interaction orchestration, not as business-logic ownership.
